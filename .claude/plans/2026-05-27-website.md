# Plan: portfolio.baretsky.net Landing Page

## Goal
A minimal website at portfolio.baretsky.net in Gzhel style (cobalt blue + white)
showing name, tagline, and two copy-able commands. A button opens a modal with a
live embedded terminal (xterm.js → ttyd → the actual Ink app on the home server).
"Done" = the page loads at portfolio.baretsky.net, both commands copy on click,
and the terminal button opens a working session of `baretsky`.

## Constraints
- Gzhel palette only: cobalt blues (#1B3D7A, #2952A3, #5B8AC4) on porcelain white (#FAFAF8)
- GSAP for all animations — no CSS transitions on animated elements
- xterm.js must connect to a real running instance of the portfolio (not a simulation)
- Vite + vanilla TS — no React framework needed
- Must work on mobile (commands readable, terminal button visible — terminal itself shows a "use desktop" message on mobile)
- SSH command stays at portfolio.baretsky.net (same subdomain, Cloudflare Tunnel handles HTTP and SSH separately)

## Assumptions
- Website lives in `website/` subfolder of the same terminal_portfolio repo
- Deploy: git push to a bare repo on the home server → post-receive hook builds + copies to nginx root
- Home server runs nginx; ttyd serves the terminal WebSocket at localhost:7681
- Cloudflare Tunnel routes portfolio.baretsky.net HTTP → nginx:80; SSH TCP tunnel stays separate
- Terminal opens in a modal overlay; each visitor gets an independent `baretsky` process
- Subtle Gzhel botanical SVG motif in one corner (hand-crafted SVG, no image files)

## Steps

1. **Scaffold `website/` in the same repo**
   - `npm create vite@latest website -- --template vanilla-ts`
   - Install: `gsap`, `@xterm/xterm`, `@xterm/addon-fit`, `@xterm/addon-web-links`
   - Verify: `npm run dev` renders Vite default page at localhost:5173

2. **HTML structure**
   - Single screen: Gzhel SVG motif (top-right corner), name + title, tagline, two command cards, terminal button, modal overlay (hidden)
   - Verify: all elements in DOM, no horizontal scroll at 320px or 1440px

3. **Gzhel CSS**
   - CSS custom properties for the palette
   - Command cards: white box, cobalt border, monospace font, copy icon
   - Botanical SVG: few paths, cobalt-mid on transparent — decorative, not distracting
   - Verify: cobalt-on-porcelain feel, readable at a glance

4. **GSAP entrance animations + theme toggle**
   - Load: name fades+slides up → tagline staggers in → command cards slide from below → button pulses once
   - Hover: card scale + border brightens (quickTo)
   - Modal open/close: overlay fade + panel slides up from bottom
   - Theme toggle: sun/moon icon top-right; GSAP animates CSS custom properties on :root for smooth color crossfade; theme saved to localStorage; applied before first paint to avoid flash
   - Verify: 60fps, no layout shift, theme persists on refresh, no white flash on dark mode load

5. **Copy-to-clipboard**
   - Click → copy → label flips to "Copied!" for 1.5s (GSAP text swap)
   - Fallback: `document.execCommand('copy')` for HTTP
   - Verify: both commands copy in Chrome + Firefox

6. **xterm.js terminal modal**
   - On open: new Terminal + FitAddon, WebSocket to `wss://portfolio.baretsky.net/ws`
   - On mobile (window.innerWidth < 768): show "Open on desktop to use the terminal" instead
   - On close: dispose terminal + close socket
   - Verify: modal opens, portfolio renders, keyboard works, close cleans up

7. **Home server: ttyd systemd service**
   - Install ttyd binary to `/usr/local/bin/ttyd`
   - systemd unit: `ttyd -p 7681 -W /usr/local/bin/portfolio-run`
   - Verify: `curl http://localhost:7681` returns ttyd HTML

8. **Home server: nginx config**
   - Serve `/var/www/portfolio/` as static root
   - `location /ws` → proxy_pass to `http://localhost:7681/ws` with WebSocket upgrade headers
   - Verify: static page serves on :80, WebSocket handshake at `ws://localhost/ws` succeeds

9. **Git-based deploy: bare repo + post-receive hook**
   - On home server: `git init --bare /repos/terminal-portfolio.git`
   - post-receive hook: pull worktree → `cd website && npm ci && npm run build` → `rsync dist/ /var/www/portfolio/`
   - On dev machine: `git remote add homeserver user@homeserver:/repos/terminal-portfolio.git`
   - Verify: `git push homeserver main` triggers build and updates the site

10. **Cloudflare Tunnel update**
    - Add ingress: `portfolio.baretsky.net → http://localhost:80`
    - SSH TCP tunnel stays as-is
    - Verify: `https://portfolio.baretsky.net` loads over Cloudflare; `ssh portfolio@portfolio.baretsky.net` still works

## Rollback
- nginx: revert server block, `nginx -s reload`
- ttyd: `systemctl disable --now ttyd`
- Deploy: `git revert` on main, push to homeserver remote triggers rebuild
- Cloudflare: remove HTTP ingress rule from tunnel config

## Out of scope
- Content beyond the one-screen landing
- Dark mode
- SEO / Open Graph beyond basic meta tags
- Authentication on the terminal
- Animations on scroll (everything is above the fold)
